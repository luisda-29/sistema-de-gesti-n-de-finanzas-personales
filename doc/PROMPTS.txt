promt 1 = 

Crea un CRUD en JavaScript Vanilla que use LocalStorage como base de datos.

Requisitos obligatorios:

1. El ID debe generarse automáticamente con base en la longitud actual del array de objetos guardado en LocalStorage.
   - Ejemplo: si hay 3 objetos, el nuevo ID será 4.
   - Debe evitar colisiones si se eliminan elementos (puedes validar o ajustar).

2. Las funciones deben ser simples y claras.
   - Máximo 20 líneas de código por función.
   - Sin sobreingeniería ni patrones complejos.

3. Las funciones deben ser independientes de los datos entrantes.
   - Deben recibir parámetros dinámicos.
   - No deben depender de una estructura fija (los campos los definiré después).

4. Usar únicamente JavaScript Vanilla.
   - No usar frameworks ni librerías externas.

5. Debe incluir las 4 operaciones:
   - create()
   - readAll()
   - update()
   - delete()

6. Usar buenas prácticas básicas:
   - Manejo de errores simple (try/catch si aplica).
   - Validar existencia en LocalStorage.
   - Convertir correctamente JSON ↔ Array.

7. El código debe estar comentado para fácil comprensión (nivel principiante).

Entrega el resultado en un solo archivo JS listo para usar.
